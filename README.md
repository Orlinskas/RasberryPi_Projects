# RaspberryPi_Projects

Raspberry Pi projects.

## robot_prome_v1

Autonomous LLM-driven robot: camera (Picamera2/OpenCV), ultrasonic sensor, and Ollama decide actions (move, turn, light, play). Modular architecture via JSON: vision → state.json → brain → command.json → controller. See [robot_prome_v1/README.md](robot_prome_v1/README.md) for details.
